# PilaHub - Queue Management System

A modern queue management system built with React Native and Firebase.

## Features
- Real-time queue updates
- QR code scanning to join queue
- Admin dashboard with monitor mode
- Haptic feedback notifications

## Tech Stack
- React Native / Expo
- Firebase Firestore
- React Navigation

## Setup
```bash
npm install
npm start
```
